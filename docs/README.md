# Documentation

This directory contains all project documentation.

## Active Documentation

- **[FEATURES.md](./FEATURES.md)** - Complete features documentation (single source of truth)
  - All implemented features
  - Implementation details
  - Usage instructions
  - Component specifications

## Archive

The `archive/` directory contains historical documentation that has been consolidated into FEATURES.md. These are kept for reference but are no longer maintained.

## Maintenance

When adding new features:
1. Update `FEATURES.md` with the new feature section
2. Update `../CHANGELOG.md` with the version change
3. Update `../README.md` if needed

**Rule**: Keep documentation in ONE place. Don't create multiple files for the same information.
